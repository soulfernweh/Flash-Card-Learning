# 🧠 Shichida Kids - Right Brain Training

🌐 **[Play Now → https://soulfernweh.github.io/Flash-Card-Learning/](https://soulfernweh.github.io/Flash-Card-Learning/)**

A browser-based learning app for kids inspired by the **Shichida Method** of right-brain education. Designed for mobile/tablet use, hosted on GitHub Pages.

## 🎮 Activities

| Activity | Description |
|----------|-------------|
| ⚡ Flash Cards | Rapid image/word exposure with adjustable speed |
| 🧩 Memory Grid | Memorize and recall positions of items in a grid |
| 🔗 Linking Memory | Remember a sequence of items shown in a story |
| 🎨 Mandala | Memorize and reproduce a colored pattern |
| 🔮 ESP Game | Intuition training - guess the hidden card |
| 🔢 Math Dots | Subitizing - quickly count dots shown briefly |
| 🎯 Identify! | Quiz game - identify flags, animals, and vehicles |

## 👶 Age Groups

- **1-3 years (Toddler)** — Simpler content, fewer items, longer display times
- **3-6 years (Preschool)** — More items, moderate difficulty
- **6+ years (Advanced)** — Complex content, faster pace, larger grids

## 🧠 About the Shichida Method

The Shichida Method, developed by Professor Makoto Shichida, focuses on right-brain education through rapid exposure, visualization, and memory training. Key principles include:

- **Speed**: Flash cards shown rapidly to engage the right brain
- **Repetition**: Regular short sessions for best results
- **Photographic memory**: Grid and pattern recall exercises
- **Linking memory**: Creating stories to remember sequences
- **Intuition**: ESP-style games to develop pattern recognition
